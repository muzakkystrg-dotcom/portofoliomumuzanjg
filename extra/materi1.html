Database dan DBMS

  • Database:

    Database adalah sebuah sistem yang dirancang untuk mengelola dan menyimpan data secara terstruktur, sehingga data dapat diakses dan dikelola dengan mudah. Dalam kehidupan sehari-hari, kita menggunakan basis data tanpa disadari. Contoh sederhana adalah ketika kita menyimpan kontak di ponsel, sistem menyimpan data kontak dalam format yang dapat kita akses kembali kapan saja. Produk basis data yang sering digunakan meliputi MySQL, Oracle Database, Microsoft SQL Server, dan MongoDB.

  • DBMS (Database Management System):

    Software yang digunakan untuk mengelola database supaya kita bisa menyimpan, mencari, mengubah, dan menghapus data dengan aman. Contoh DBMS: MySQL, Oracle, PostgreSQL.
