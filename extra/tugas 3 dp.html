Pengalaman Saya Selama Mengikuti Pembelajaran Database

Pengalaman saya selama mengikuti pembelajaran database adalah

Saya dapat memahami jenis jenis data dan memahami bagaimana data data disimpan, disortir, diatur, dan dikelola. Awalnya saya sedikit kebingungan memahami istilah istilah seperti table, primary key, dan field,tetapi setelah di jelaskan secara bertahap oleh bu ike saya perlahan dapat memahami apa itu database. Selain itu, saya juga belajar bagaimana membuat query sederhana menggunakan SQL, seperti menampilkan data, menambah data baru, menghapus, maupun mengedit data. Dari situ saya mengerti bahwa database bukan hanya sekadar tabel, tetapi merupakan sistem yang sangat terstruktur dan bermanfaat.


Pengalaman yang paling berkesan waktu mengikuti pembelajaran adalah

Saat membuat database sederhana sendiri. mulai dari mendesign tabel sampai menghubungkan relasi antar tabel. Dari situ saya belajar bahwa database sangat diperlukan di dunia coding. bukan hanya di lingkungan sekolah tetapi database juga penting di perusahaan besar. Saya juga merasa berkesan ketika berhasil menjalankan perintah SQL pertama kali dan melihat hasilnya sesuai dengan yang saya inginkan. Hal tersebut memberikan semangat lebih untuk mendalami dunia database, karena ternyata database sangat erat kaitannya dengan aplikasi maupun website yang sering kita gunakan sehari-hari.
